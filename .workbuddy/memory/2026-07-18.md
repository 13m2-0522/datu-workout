# 2026-07-18

## 主要工作：搭建「呆兔要变生猛兔」暑假锻炼记录 App

### 项目背景
用户（小学生，即将上6年级）想在暑假记录锻炼过程（哑铃、保加利亚蹲、平板支撑等），实现增肌减脂的数据追踪。需求包含10项功能：锻炼记录、日周月年统计、体重体脂提醒、语音记录感受、分享、勋章、秒表、热量估算、数据持久化、启动封面。

### 技术决策
- 采用 Web 单页应用（HTML/CSS/JS）而非微信小程序，原因：
  - 无需注册开发者账号和下载微信开发者工具
  - 手机浏览器直接打开就能用
  - PWA 支持添加到主屏幕，体验接近原生
  - 后期可快速移植到微信小程序
- 数据持久化：localStorage（key: 'datu_workout_data'）
- 手机优先设计，底部导航栏 5 个 Tab

### 已完成的 10 项功能
1. **启动封面页**：肌肉兔子动画 + "呆兔要变生猛兔" 标题，2.6秒后自动进入主界面
2. **锻炼记录**：日期选择、10种运动项目选择、组数/次数/重量输入、热量估算
3. **秒表计时**：开始/暂停/重置，计时结果可一键填入记录页
4. **数据统计**：日/周/月/年四个维度，Canvas 自绘柱状图和折线图
5. **体重体脂提醒**：锻炼后自动弹出提醒，引导记录身体数据
6. **语音输入**：支持录音（浏览器 MediaRecorder API）
7. **分享功能**：生成分享图片 + 复制文字到剪贴板
8. **勋章系统**：10种勋章（青铜兔/白银兔/黄金兔/钻石兔/烈焰兔/满级兔/闪电兔/全能兔/打卡兔/燃烧兔），条件达成自动点亮
9. **热量估算**：基于 MET 代谢当量法计算消耗
10. **数据持久化**：localStorage 保存 + 支持 JSON 导出/清空

### 文件结构
- `index.html` - 主应用（单文件，含全部 HTML/CSS/JS）
- `manifest.json` - PWA 配置
- `assets/rabbit_splash.png` - 启动封面兔子图（AI生成，3D可爱风）
- `assets/rabbit_cropped.png` - 完整图（去水印）
- `assets/rabbit_transparent.png` - 透明背景版
- `assets/A_super_cute_*.png` - 原始生成图
- `.workbuddy/memory/` - 开发记录
